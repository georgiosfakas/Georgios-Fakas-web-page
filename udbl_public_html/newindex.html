Uppsala DataBase Laboratory

Welcome to Uppsala DataBase Laboratory (UDBL). We are a research group in the Department of Information Technology at Uppsala University.  

UDBL conducts research on methods and theories for database technology. Our research emphasis is on investigating how Object-Oriented query processing can be used for querying and integrating data from heterogeneous databases, storage managers and other data sources in a distributed environment.

We are responsible for the Doctoral study subject area on Computer Science with specialization in Database Technology, TNDAVE04. A doctoral study plan (currently only in Swedish) can be found here. We are teaching doctoral student level courses within the programme on modern database technology. For the undergraduate education we are responsible for courses in Database Design. 

Swedish textbook on database technology: Databasteknik

In the POQSEC Project (supported by VR) we devlop a scalable and deistributed query processor utilizing the peer-oriented GRID infrastructure NorduGRID. This project is an UPPMAX project.

In the GSDM project (supported by Vinnova) we develop a stream database query manager for queries over high volume scientific queries for space applications. This project is in cooperation with IRFU and ASTRON.

We particitape in ODBMS.ORG which is a portal providing the most up-to-date collection of free materials on object database technology on the Internet.

We are members of the NorFA network Wireless Information Management (WIM), which is a research training network on position aware e-services involving researchers from six universities in Denmark, Finland, Lithuania, Norway, and Sweden. A Nordic WIM workshop was arranged in Uppsala in September 2004.

A large part of the research is based on the umbrella project Amos II - Active Mediators Object System. Amos II serves as an umbrella project where subprojects contribute parts of a general vision. Theoretical results are verified by incorporation in the Amos II architecture and prototypes. Amos II uses a functional data model for database representation, querying, and integration of heterogeneous data. There is an up-to-date overview of Amos II in

P.Gray, L.Kerschberg, P.King, and A.Poulovassilis (eds.): Functional Approach to Data Management - Modeling, Analyzing and Integrating Heterogeneous Data, Springer, ISBN 3-540-00375-4, 2003.     

We have developed the research programme IEIS, Integrated Engineering Information Systems, which serves as one focal point for our research. Its purpose is to solve database integration problems in the engineering domain.

We participated in the project Personalized Access to Distributed Learning Repositories (PADLR) which is part of the Wallenberg global learning network research program on distributed collaborative learning innovations. Within PADLR we have developed PSELO, a semantic web based query processing system for searching learning material from a peer-to-peer based educational infrastructure.

We participated in the NEUROGENERATOR project which is an EC funded project on a database generator for the Neuroimagining community. Other partners are KTH, KI (Stockholm), FORWISS, and Active Knowledge.

We cooperate with formerly Alzato Inc., now MySQL Cluster, on high performance parallel database technology.
